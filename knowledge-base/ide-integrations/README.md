# IDE & Tool Integrations

## What You Can Connect

Cost Katana integrates with the tools you already use: Cursor, GitHub, Vercel, and ChatGPT. You get usage tracking, chat assistance, and workflow automation in one place.

## Cursor Extension

- **Track usage** – AI requests from Cursor are tracked automatically
- **Optimize prompts** – Get suggestions to reduce cost while keeping quality
- **Analyze code** – Use Cost Katana’s chat to analyze your code with cost context
- **Create projects** – Set up new projects with Cost Katana from Cursor
- **Workspace setup** – Connect your workspace so usage is tied to the right project

## GitHub

- **Connect repos** – Link GitHub so Cost Katana can analyze your code
- **In chat** – Ask to create PRs, list PRs, create issues, or explore your codebase
- **Webhooks** – Repo events (e.g., pushes, PRs) can trigger Cost Katana workflows

## Vercel

- **Connect projects** – Link Vercel to manage deployments from Cost Katana
- **In chat** – Ask to deploy, list projects, or view deployment logs
- **Auto-create** – Cost Katana can create Vercel projects when deploying from GitHub

## ChatGPT

- **Cost Katana plugin** – Use Cost Katana from ChatGPT for usage and cost insights
- **Actions** – Structured actions for tracking, analytics, and recommendations

## FAQ

**How do I install the Cursor extension?**  
Install from the Cursor marketplace or follow the setup guide in the Cost Katana dashboard.

**Do I need separate connections for GitHub and Vercel?**  
Yes. Connect each service in the Integrations section. Once connected, you can use them together in chat.

**Can I track usage from multiple IDEs?**  
Yes. As long as the extension or SDK is configured with your API key, usage is tracked to your account.
